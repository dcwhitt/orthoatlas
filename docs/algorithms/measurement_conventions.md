# OrthoAtlas measurement conventions

- Elbow extension: 0° is full extension. Positive values indicate extension deficit. Hyperextension may be negative only if manually confirmed.
- Knee extension: 0° is full extension. Positive values indicate flexion contracture/extension deficit. Hyperextension may be negative only if manually confirmed.
- Shoulder elevation is measured relative to the trunk/hip-shoulder axis rather than raw image vertical.
- Hip ROM is calculated from trunk/hip/knee pose landmarks and should be interpreted in the context of camera position.
- Ankle dorsiflexion/plantarflexion is a tibia-foot proxy and should be treated as a screening ROM value.
- No video is uploaded or stored. Exports contain measurements and, for full JSON/long CSV, landmark-derived frame data.
